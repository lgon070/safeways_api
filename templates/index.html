SafeWays Safepath API

Beta API that finds the safest path for pedestrians and bicyclists, made for City of Los Angeles residents using LA City Collision Data, Google's Directions API, and Google's LocationOnEdge and/or ContainsLocation algorithm

How to use this API?

Parameter Description Example Required
origin latitude and longitude of the origin in decimal degrees separated by a comma. origin=34.052235,-118.243683 required
destination latitude and longitude of the destination in decimal degrees separated by a comma. destination=34.059238,-118.279068 required
method method of navigation (walking, bicycling, or driving) method=walking or method=bicycling or method=driving required
use_edge indicates whether API will use ContainsLocation (0) or LocationOnEdge (1) algorithm use_edge=0 or use_edge=1 optional (will default to use_edge=1 if parameter not provided)
tolerance indicates the tolerance level of the LocationOnEdge algorithm (does not affect ContainsLocation tolerance=30 optional (will default to 20 if parameter is not provided or falls outside 5[inclusive] - 150[inclusive])

Full Example of a Valid API Call

https://polar-hollows.wl.r.appspot.com/safepath?origin=34.053715,-118.242653&destination=34.059238,-118.279068&method=walking
